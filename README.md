# snapgram
인스타그램 클론 코딩

### version 1
- 회원
    - 로그인
    - 회원가입
    - 비밀번호 찾기
    - 아이디 찾기
    - 팔로우 기능
- 메인 페이지
    - 팔로우 한 사람들의 게시물 목록
- 개인 페이지
    - 내 게시물 목록
    - 내 정보 조회(게시물 수, 팔로우 수, 팔로워 수, 계정 아이디)
- 상세 페이지
    - 게시글 조회
    - 좋아요(숫자, 누가 좋아요 했는지)
    - 댓글(숫자, 댓글 단 사람 계정)
- 검색 기능
    - 계정 검색
